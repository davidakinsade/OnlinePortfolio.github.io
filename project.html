Group Project In Agile Software Development

Sep 2020 - Apr 2021

A software development based project I completed of recent was with a group where our objective objective was to create and implement a newspaper delivery system for a newsagent. We had to look at aspects such as delivery routes, products, customers receiving products, invoices, admin accounts and orders.
Personally, it was nice to learn new aspects of coding with a group and I felt that it was informative working with features like the backlog and the repository hosting service. Our group also got a better understanding of working with boundary values on each class and test coverage.
I feel that this exercise is a huge benefit for me as a software developer as I would have an idea of how to work with clients and how to meet their demands.
